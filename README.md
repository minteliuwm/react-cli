# custom-react-cli

## Install Cli
```
npm install custom-react-cli -g
```

### Generate a project
```
custom-react-cli init [projectName]
```

## Project setup
```
cd [projectName]
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
